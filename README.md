
## Getting Started

First, run the install scripts:<br/>

yarn install or npm  install<br/>

once the installation is done:<br/>

For production build:<br/>
yarn build <br/>
or npm run build<br/>
yarn start<br/>
or npm run start<br/>

For Dev mode:<br/>
yarn dev<br/>
or npm run dev<br/>

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
